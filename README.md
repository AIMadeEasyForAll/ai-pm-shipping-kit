# The AI PM Shipping Kit

## Scope, evaluate, and ship AI features with artifacts you can put in front of engineering and leadership **tomorrow**.

Built by [Mani Grewal](https://productmanagementwithmani.substack.com) · For working product managers shipping AI features.

---

Connecting a model to your documents sounds simple. Defining what *good* looks like and proving it works is where most teams get stuck. This kit gives you the operating tools senior AI PMs use so you stop shipping on "it feels right" and start shipping on evidence.

You don't get a course. You get working Claude Skills and filled-in artifacts that do the job.

## What's inside

| Asset | What it does for you |
|---|---|
| **RAG Scoping Skill** | Paste your feature context, get a complete scoping doc: retrieval design, data boundaries, failure modes, and a definition of done eng will accept. |
| **Eval Plan Skill** | Turns a vague feature into a measurable one. Five metrics that matter, before-and-after-ship rubrics, and a test set plan. |
| **Scoping artifact template** | The fill-in doc the Skill produces, so you can edit and share it directly. |
| **Eval rubric template** | The rubric that replaces subjective sign-off in your reviews. |

## How the kit works

``` (mermaid live) flowchart LR
    A[Your feature idea] --> B[RAG Scoping Skill]
    B --> C[Scoping doc<br/>eng accepts]
    C --> D[Eval Plan Skill]
    D --> E[Eval rubric +<br/>test set]
    E --> F[Ship with evidence]
    F --> G[Defend to<br/>leadership]
```

## Quick start

1. Open Claude (or Claude Code).
2. Copy a Skill from `/skills` into your Claude Skills, or paste its contents into a chat.
3. Fill in the context block at the top with your real feature.
4. Run it. Edit the output. Take it to your standup.

## Free vs full

This public repo is the **lite** version: the RAG Scoping Skill and its artifact template.

The **full kit** adds the Eval Plan Skill, the AI Opportunity Scorer, the launch-readiness checklist, and worked examples on real features. → [Get the full kit](https://productmanagementwithmani.substack.com)
